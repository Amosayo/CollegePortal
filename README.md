# College Portal

A web-based college portal built with Python Django that serves both new students who want to register and returning students who need to register for their courses.

# Key Features

New student account registration
Secure login for students and administrators
Course registration for returning students
Profile management for students
Admin panel for managing courses, students, and departments
Email confirmation (optional feature)
Unit tests for core functionality

# Tech Stack

- **Backend**: Python 3.13.3, Django 5.2.7
- **Database**: SQLite (development)
- **Frontend**: HTML5, CSS3, Bootstrap 5
